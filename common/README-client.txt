OffGrid — Offline Penetration Testing Platform
================================================

HOW TO START:

1. Install VMware Player (free): https://www.vmware.com/products/workstation-player.html
   OR VirtualBox (free):         https://www.virtualbox.org/wiki/Downloads

2. Open VMware Player → File → Open → select the .vmx file in this folder
   OR VirtualBox → New → Use existing disk → select the .vmdk file

3. Start the VM.

4. Login: kali / kali

That's it. All tools are pre-installed. No internet required.
